# TargetBulkUpload
Bulk upload data to target dcc

## Install python3

## Prepare library (if necessary)
```
pip3 install xlrd
```

## How to use it
1. Obtain your personal token.
2. Fill in the excel template.
3. Run it with command:
```
python3 submission.py -k <token string> -t V2 -x <excel file>
```
